01. Desafios de Flexbox Básico:
	1) Faça uma lista de itens que se alinha horizontalmente usando Flexbox.
	2) Crie um layout de cards que se ajusta automaticamente ao número de cards usando Flexbox.
	3) Estilize um menu de navegação horizontal simples usando Flexbox.
	4) Faça um layout de duas colunas que se ajusta ao tamanho da tela usando Flexbox.
	5) Crie um layout de galeria de imagens que se alinha em linhas usando Flexbox.

02. Desafios de Grid Básico:
	1) Faça um layout de 3 colunas usando Grid Layout.
	2) Estilize uma seção da página para ter uma grade de imagens usando Grid Layout.
	3) Crie um layout de card com Grid Layout que se ajusta ao tamanho da tela.
	4) Faça um layout de 4 quadrados que se ajusta ao tamanho da tela usando Grid Layout.
	5) Crie um layout de página com cabeçalho, conteúdo em duas colunas e rodapé usando Grid Layout.

03. Desafios de Responsividade Básica:
	1) Faça um menu de navegação que se transforma em um menu hamburguer em telas menores.
	2) Crie um layout de card que se ajusta ao tamanho da tela em dispositivos móveis.
	3) Estilize um formulário para ter um layout adequado em dispositivos móveis.
	4) Faça um layout de duas colunas que se transforma em uma coluna em dispositivos móveis.
	5) Crie um layout de página que se adapta ao tamanho da tela em dispositivos móveis.

04. Desafios de Animações e Transições Básicas:
	1) Adicione uma animação ao hover de um botão para mudar a cor de fundo suavemente.
	2) Faça uma animação para um elemento se deslocar da esquerda para a direita ao ser clicado.
	3) Estilize um elemento para ter uma transição suave ao mudar de cor ao passar o mouse.
	4) Crie um botão que ao ser clicado, cresce de tamanho com uma animação.
	5) Faça um elemento girar continuamente em loop usando keyframes.

05. Desafios de Pseudo-classes e Pseudo-elements Básicos:
	1) Estilize um link para ter uma cor diferente ao ser clicado usando a pseudo-classe :active.
	2) Faça um card ter um efeito de sombra ao passar o mouse sobre ele usando a pseudo-classe :hover.
	3) Estilize um botão para ter uma seta no final usando a pseudo-element ::after.
	4) Faça uma lista de itens ter marcadores personalizados usando a pseudo-element ::before.
	5) Crie um layout de cartão com uma imagem e um título, e estilize o título usando a pseudo-element ::first-letter.

06. Desafios de Forms Básicos:
	1) Estilize um formulário de login para ter um visual moderno e agradável.
	2) Faça um campo de busca ter um ícone de lupa no lado esquerdo usando CSS.
	3) Estilize um formulário de cadastro para ter campos com bordas arredondadas.
	4) Faça um botão de envio de formulário ter um efeito de loading ao ser clicado.
	5) Crie um layout de formulário responsivo com rótulos flutuantes sobre os campos.

07. Desafios de Media Queries Básicas:
	1) Faça um layout de 3 colunas se transformar em uma coluna em telas menores usando media queries.
	2) Estilize um menu de navegação para se transformar em um menu hamburguer em telas menores.
	3) Crie um layout de galeria de imagens que se ajusta ao tamanho da tela usando media queries.
	4) Faça um layout de cards se ajustar em uma coluna em telas menores usando media queries.
	5) Estilize um formulário para ter um layout adequado em telas menores usando media queries.

08. Desafios de Cores e Estilos Básicos:
	1) Crie um esquema de cores para o site usando variáveis CSS.
	2) Faça um botão ter um gradiente de cores suave ao passar o mouse.
	3) Estilize um elemento para ter um efeito de sombra em forma de borda usando CSS.
	4) Faça uma seção da página ter um efeito de paralaxe ao rolar usando CSS.
	5) Crie um layout de página com uma seção de fundo gradiente e texto em camadas.

09. Desafios de Layout Responsivo Básico:
	1) Faça um menu de navegação que se transforma em um menu dropdown em telas menores.
	2) Estilize um carrossel de imagens para se adaptar ao tamanho da tela em dispositivos móveis.
	3) Crie um layout de blog responsivo com posts em forma de cards que se ajustam ao tamanho da tela.
	4) Faça um layout de página com uma seção de destaque que se transforma em uma barra lateral em telas menores.
	5) Estilize um elemento para ter um efeito de gradiente em texto ao passar o mouse.

1) Desafios de Layout:

	1. Desafio de Layout 3D com Transformações Complexas:
		1.1 Crie um layout 3D usando transformações CSS complexas, como rotação, escala e inclinação, para criar uma apresentação visualmente impressionante e dinâmica.

	2. Desafio de Layout com Animações de Partículas:
		2.1 Crie um layout com animações de partículas usando CSS e JavaScript avançado, onde as partículas se movem de forma fluida e interativa em resposta à interação do usuário.

	3. Desafio de Layout com Manipulação de Imagens em Tempo Real:
		3.1 Desenvolva um layout que permita ao usuário manipular imagens em tempo real usando filtros CSS avançados e manipulação de pixels por meio de JavaScript, proporcionando uma experiência interativa e envolvente.

	4. Desafio de Layout com Integração de Realidade Aumentada:
		4.1 Crie um layout que integre elementos de realidade aumentada usando tecnologias como WebXR e WebGL, permitindo aos usuários interagir com objetos virtuais em um ambiente 3D dentro do navegador.

	5. Desafio de Layout com Animations-driven Scroll:
		5.1 Desenvolva um layout onde a rolagem da página é conduzida por animações CSS, proporcionando uma experiência de navegação suave e cinematográfica.
